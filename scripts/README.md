# Scripts

This folder is reserved for the reproducible analysis pipeline.

Planned entry points:

- `simulate.R` for the simulation engine
- `run_simulation_grid.R` for executing the full grid and saving results
- `import_nhanes.R` for importing the NHANES case study dataset
- `import_diabetes.R` for importing the diabetes benchmark dataset
- `nhanes/eda_nhanes.R` for NHANES exploratory analysis
- `nhanes/analyze_age_decade.R` for NHANES subgroup regression stability by age decade
- `analyze_real_data.R` for dataset-specific modeling
- `make_figures.R` for figures and tables
- `utils.R` for shared helpers
